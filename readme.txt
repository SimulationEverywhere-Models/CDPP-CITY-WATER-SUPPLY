1. Open CD++ Builder

2. Create a new CD++ project ( CD++ perspective )

3. Unzip the zip the fsm.zip to CD++ Builder working folder.

4. Compile the project

5. Simulate models .ma files provided. There are also .bat files that can be run from command line. 
Attention! When you run city_water.ma model with city_water.ev in CD++ Builder you must provide a time limit, 
otherwise you end up with a huge log file. The time limit is specified in the city_water.bat file.

6. The results of simulation are saved in *.out and *.log files.
